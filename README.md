# snakebyte

Apple ][ GR 느낌의 단순한 터미널 Snake 게임입니다.

## 실행

```bash
python3 snakebyte.py
```

## 조작

- `i`: 위
- `m`: 아래
- `j`: 왼쪽
- `k`: 오른쪽
- `q`: 종료
- `r`: 게임오버 후 재시작

## 규칙

- 게임 시작 시 뱀 길이는 3입니다.
- 화면 내 랜덤한 점(`.`)을 먹으면 길이가 1씩 증가합니다.
- 벽 또는 자기 몸에 머리가 닿으면 게임오버입니다.

## 참고

- 터미널 창이 너무 작으면 게임 대신 크기 부족 안내 메시지를 표시합니다.
- 권장 최소 크기: `42 x 29` (가로 x 세로)
