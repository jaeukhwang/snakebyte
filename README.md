# snakebyte

Apple ][ GR 느낌의 **별도 윈도우 40x40 매트릭스 그래픽 모드** Snake 게임입니다.

## 실행

```bash
python3 snakebyte.py
```

## 조작

- `i`: 위
- `m`: 아래
- `j`: 왼쪽
- `k`: 오른쪽
- `q`: 종료
- `r`: 게임오버 후 재시작

## 규칙

- 게임 시작 시 뱀 길이는 3입니다.
- 40x40 매트릭스 안에서 랜덤 점(먹이)을 먹으면 길이가 1씩 증가합니다.
- 벽 또는 자기 몸에 머리가 닿으면 게임오버입니다.
- 벽에 붙어 생성된 먹이(빨간색)는 난이도 높은 먹이이며, 먹으면 뱀 길이는 1 줄고 점수는 일반 먹이처럼 1 증가합니다.

## 구현 메모

- 터미널 텍스트 렌더링이 아니라 `tkinter` 캔버스 기반 별도 창 렌더링입니다.
- 내부 논리 해상도는 `40 x 40`, 각 셀은 `12 x 12` 픽셀입니다.
